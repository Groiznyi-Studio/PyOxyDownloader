Classes:
1)Core - The api itself
    Functions:
      1)GetMainUrl() - get url(Using input())
      2)GetParam(Param) - Param type(str) - return data from __Params(Core.__Params)
      3)ClearParam(Param) - Param: type(str) - clears the specified parameter from(Core.__Params)
      
      4)GetHtmlCodePage(Url, Content = False) - Url: type(str), Content: type(bool) - downloads the page code and stores it in __Params["HtmlCodePage"],
      if Content == True then save the file to __Params["HtmlCodePage"],
      if an error occurs, throws a CoreException.
      
      5)WriteFile(Path, NameFile) - Path: type(str) endline '\', NameFile: type(str) - saves the file that was received by the link,
      throws a CoreException when an error occurs
      
      6)GetNextUrl(Step) - Step: type(int), if Step == 1 - get the second download link(Oxy Cloud only (Parser for Oxy Cloud)) and saved in __Params["SecondUrl"],
      if Step == 2 - get the final download link(Oxy Cloud only (Parser for Oxy Cloud)) and saved in __Params["FinalUrl"],
      throws a CoreException when an error occurs
