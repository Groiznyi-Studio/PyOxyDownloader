# PyOxyDownloader
PyOxyDownloader - Script for fast and safe download of files from the oxy cloud website

# Demands
1)Python 3.3+
2)Windows Vista - 10

# How to use
# Single load
1)In the field enter url to the file with oxy cloud and press enter

2)Warning1: The file will be saved in the current script folder

3)Example: url oxy.cloud: https://oxy.cloud/d/SKqc and press enter

4)Ready!!!

# Multi downloads mode
1)In the field enter urls to the file with oxy cloud and press enter

2)Warning1: The files will be saved in the current script folder

3)Warning2: The main thing is to put ":" after each url and at the end of the line too!!!

4)Example: url oxy.cloud: https://oxy.cloud/d/SKqc:https://oxy.cloud/d/Gbo: and press enter

5)Ready!!!
